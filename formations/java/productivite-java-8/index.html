Formation Productivité avec Java 8+

Maitriser les nouveautés de Java 8+

Présentation

Adopter l'approche fonctionnelle avec Java.

Objectifs

La formation Productivité avec Java 8+ vise l'acquisition des compétences suivantes :

  • Découvrir les nouveautés de Java 8+
  • Comprendre ce que Java 8 apporte en terme de productivité

Prérequis

  • Connaissance du langage Java

Durée

3 jours

Méthodologie

  • La formation est composée d'une suite d'exposés (50%) et travaux pratiques (50%)
  • Le logiciel MicroSoft Teams est utilisé pour les formations à distance

Adaptations

  • Toutes nos formations peuvent s'adapter à vos besoins à la suite d'un audit du besoin de formation. Les modalités décrites sont décrites ici

Validation des acquis

  • La validation de l'acquisition des compétences est réalisée par le formateur pendant la formation.
  • Sur la base d'une grille d'évaluation et de critères communiqués aux stagiaires, le formateur évalue les compétences de chacun d'eux.

Lieu de la formation

La formation se déroule :
  • soit dans vos locaux.
  • soit à distance via l'utilisation du logiciel MicroSoft Teams
Pour un public en situation d'handicap, nous vous invitons à nous contacter pour déterminer les modalités adéquates.

Tarifs

  • 1 000 € HT pour une formation à distance et pour 4 participants maximum
  • 1 300 € HT pour une formation dans vos locaux et pour 4 participants maximum
  • Un devis personnalisé sera établi pour les cas non prévus

Délai d'accès

Un devis et une proposition de dates vous seront communiqués au plus tard 24h après votre demande.

Indicateurs de résultat

Les indicateurs de résultats seront publiés à la suite des prochaines formations. Les indicateurs planifiées
  • Moyenne des objectifs atteints : -
  • Taux de satisfaction global des stagiaires: -
  • Nombre de stagiaires formés : -

Certification professionnelle

Cette formation ne donne pas lieu à une certification professionnelle. Une attestation de formation vous sera délivrée.
